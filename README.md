# NLP Chatbot using Python and NLTK

A simple rule-based chatbot built using Python and the Natural Language Toolkit (NLTK). This project demonstrates the basics of Natural Language Processing (NLP) by creating an interactive chatbot that can respond to predefined user inputs.

## 📌 Project Overview

This project simulates a conversational chatbot that interacts with users through the command line.

The chatbot uses pattern matching to recognize user inputs and generate appropriate responses for common conversations such as greetings, introductions, and basic questions.

This project is designed for beginners who want to learn the fundamentals of Natural Language Processing and chatbot development using Python.

## 🚀 Features

* Interactive command-line chatbot
* Rule-based conversation system
* Pattern matching using regular expressions
* Multiple predefined responses
* Greeting recognition
* Basic question answering
* Exit command support
* Lightweight and beginner-friendly implementation

## 🛠️ Technologies Used

* Python
* NLTK (Natural Language Toolkit)

## ▶️ How to Run the Project

### 1. Clone the repository

```bash
git clone https://github.com/Harshit-Swain-25/ChatBot-Using-NLP.git
```

### 2. Navigate to the project folder

```bash
cd ChatBot-Using-NLP
```

### 3. Install the required dependency

```bash
pip install nltk
```

### 4. Run the chatbot

```bash
python chatbot_nlp.py
```
