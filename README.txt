=======================
   ChatBot Using NLP
=======================


Description:-
This is a simple rule-based chatbot built using Python and the NLTK (Natural Language Toolkit) library.
It can respond to greetings, basic questions, and simple conversations.



Features:-

1. Responds to greetings like “hi”, “hello”, or “hey”.

2. Can answer basic questions such as “what is your name?” or “how are you?”.

3. Uses regex patterns to match user input and reply accordingly.

4. Exits when you type “bye”.



How to Run:-

1. Open a terminal or VS Code.

2. Make sure Python (3.8 or later) is installed.

3. Run the Python file:

    python3 chatbot_mlp.py
